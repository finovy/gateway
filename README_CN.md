该项目在Spring Cloud Gateway的基础上进行了拓展，构建成了一个高性能网关。它不仅提供了代理、路由、负载平衡、健康检查、身份验证等功能，还可以作为系统流量的入口层。同时，我们还实现了在配置中心自定义路由文件的功能，从而实现了路由的动态刷新。
